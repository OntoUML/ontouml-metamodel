# OntoUML Metamodel

The OntoUML Metamodel is used for the JSON ([OntoUML-Schema](https://github.com/OntoUML/ontouml-schema)) and TTL serializations ([OntoUML Vocabulary](https://github.com/OntoUML/ontouml-vocabulary)).
